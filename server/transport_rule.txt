서버와 클라이언트가 통신을 주고 받을 때 전송하는 형식

# server TO client
1. 게임 종료 시
"FIN\n#1\n"
- #1 : 이긴 사람을 나타냄(1은 WHITE, 2는 BLACK)

2. 보드 정보 보내기
"TUR\n#1\n#2#3#4#5\n"
-#1 : 턴수를 나타냄
-#2 ~ #n : 보드판을 나타내며, 64*2의 문자로 구성. 하나의 말이 2개의 문자로 표현됨

3. 움직일 수 있는 좌표 보내기
"SEL\n#1\n#2#3#4\n"
-#1 : 턴수를 나타냄
-#2 ~ #n : 선택된 말이 움직일 수 있는 좌표를 나타내며, 2개의 문자(행열)가 하나의 좌표를 나타냄

4. 움직였는지 여부 보내기
"MOV\n#1\n"
#1 : 성공했으면 1, 실패했으면 0

5. 게임이 끝났는지 여부 보내기
"FIN\n#1\n"
#1 : 이긴 사람 여부로  white승리 시 1, black 승리 시 2

# client TO server
